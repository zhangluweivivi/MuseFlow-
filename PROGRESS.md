```

用户点击"确定"
    ↓
┌─────────────────────────────────────────────────────────────┐

│                                                              │

    ↓ (失败时)
┌─────────────────────────────────────────────────────────────┐
│  方案二：UI 注入                                              │

    ↓ (失败时)
```

### 技术实现

**方案一（API）**:
```javascript
// comfyui-api.js
async submitWorkflow(workflow, options) {
  const response = await fetch('http://127.0.0.1:8000/prompt', {
    method: 'POST',
    body: JSON.stringify({ prompt: workflow, ... })
  });
  return { promptId, number };
}
```

**方案二（UI 注入）**:
```javascript
// comfyui-preload.js
ipcRenderer.on('load-workflow', (event, workflow) => {
  app.loadGraphData(workflow);  // ComfyUI 内置 API
  showNotification('工作流已加载');
});
```

**方案三（手动）**:
```javascript
// app.js
await window.electronAPI.openProjectFolder(projectPath);
alert('请按以下步骤操作：\n1. 打开 ComfyUI...');
```
